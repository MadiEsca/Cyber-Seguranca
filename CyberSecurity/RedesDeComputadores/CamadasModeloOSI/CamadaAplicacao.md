# Camada de Aplicação

- É a camada mais próxima do usuário final, além de ser a última camada dentro do modelo OSI.
- A camada de aplicação é responsável por identificar e estabelecer a comunicação entre um destinatário e o remetente. (*Client-Server*)
- Essa é uma estrutura já conhecida por nós, e que está presente em quase todas, se não todas, as conexões de rede modernas, a famosa ***Client-server***.
  - Um exemplo disso é quando queremos acessar determinado site na Web. Nesse caso, nosso navegador (*Client*) faz uma requisição a um servidor Web (*Server*), onde os dados que queremos estão alocados.

## Exemplos

- Navegadores e Servidores Web
- E-mail Gateways
- Bancos de Dados

## Protocolos utilizados

- *`HTTP(s)`*
- *`DNS`*
- *`SMTP`*
- *`SNMP`*

## Tópicos para pesquisa

- Redes P2P (Peer-to-Peer)
